Playing around with antlr 3.5

Simple calculator written using antlr 3.5 version.
Could calc following expressions:
(1+3) - 8 * ( (-2+4) + 11 ) * 9 

Using Eclipse gallileo + antlr3.5 since was unable to make latest Eclipse + antlr 4.0 working together.
